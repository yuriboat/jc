# 编者请注意
- 在GitHub的项目README和侧边栏会显示contributors（建设者），GitHub提交记录会显示创建/更改教程的具体作者，所以：如果网络条件允许，建议使用GitHub账号快速登录，记得给GitHub账号本身设置头像并设置姓名（注意是姓名，姓名！如下图所示）；建议 使用本地账号登录 的编者在GitHub注册账号并更换头像、设置姓名（注意是姓名，姓名！如下图所示），以及设置简介（可选），并确保本地账号绑定的邮箱也绑定了GitHub。
- 你可以在[about](/about)页面「如何支持本站」一节填上你的爱发电链接。请按照以下格式添加（括号内内容替换为链接，需带"https://"）：[你的昵称]的平台(https:// ) - 岗位和主要编辑内容
-> 示例：[Harico Twilight的「爱发电」](https://afdian.net/a/harico) - 本站的创始人、主要管理者和主要建设者，负责编写绝大部分教程；本站所属企划（「百合花船」）主理人；本站所属社团（百合研社团）主编

- 已注册本地账号或登记使用GitHub账号登录的编者，请尽量在网站编辑，因为更改可以立即反映到网站。在GitHub编辑，或是clone到本地更改后再push到远程仓库，需要最多30分钟同步更改。
- 在上传图片前，注意把图片压缩到5M以下（服务端已做限制），建议转换为webp格式（不强制）；文件名改为「路径-说明/标号」，比如我想给https://jc.yuriboat.cn/test插入一张图片，那么图片文件名应为「test-01」或者「test-(这张图的作用)」。**请务必在图片的选择/上传页面点击「Pictures」，再进行上传。**
- 部分页面如果需要较长时间建设，不打算对外公布，记得在创建页面时将页面设置为「未发布」。
